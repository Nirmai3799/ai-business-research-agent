# AI Business Research Agent

AI-powered research automation workflow built using n8n and OpenAI APIs.

## Features
- AI-generated business research summaries
- Structured insights extraction
- Google Sheets integration
- Muti-channel Workflow automation
- Semantic Scholar integration
- ElevenLabs voice integration
- Conditional routing in n8n

## Tech Stack
- n8n
- OpenAI/OpenRouter
- Google Sheets
